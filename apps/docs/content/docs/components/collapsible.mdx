---
title: Collapsible
description: A single collapsible section with a trigger button and animated content panel.
---

import { CollapsibleBasic, CollapsibleDefaultOpen, CollapsibleDisabled } from '@/components/demos/collapsible-demo';

## Import

```tsx
import { Collapsible } from '@basex-ui/components';
```

## Usage

<CollapsibleBasic />

```tsx
<Collapsible.Root>
  <Collapsible.Trigger>What is BaseX UI?</Collapsible.Trigger>
  <Collapsible.Panel>
    BaseX UI is an AI-first component library built on Base UI and styled with StyleX.
  </Collapsible.Panel>
</Collapsible.Root>
```

## Default open

<CollapsibleDefaultOpen />

```tsx
<Collapsible.Root defaultOpen>
  <Collapsible.Trigger>Project details</Collapsible.Trigger>
  <Collapsible.Panel>
    This panel starts open. Click the trigger to collapse it.
  </Collapsible.Panel>
</Collapsible.Root>
```

## Disabled

<CollapsibleDisabled />

```tsx
<Collapsible.Root disabled>
  <Collapsible.Trigger>Locked section</Collapsible.Trigger>
  <Collapsible.Panel>This content is hidden and cannot be revealed.</Collapsible.Panel>
</Collapsible.Root>
```

## Parts

| Part | Description |
|------|-------------|
| `Root` | Container managing open state |
| `Trigger` | Button that toggles the panel |
| `Panel` | Collapsible content area |

## Props

### Root

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `open` | `boolean` | — | Controlled open state |
| `defaultOpen` | `boolean` | `false` | Initially open |
| `onOpenChange` | `(open) => void` | — | Called on change |
| `disabled` | `boolean` | `false` | Disabled |
