title: Avatar
description: A circular image or fallback representing a user or entity.
---

import { AvatarBasic, AvatarFallback } from '@/components/demos/avatar-demo';

## Import

```tsx
import { Avatar } from '@basex-ui/components';
```

## Usage

<AvatarBasic />

```tsx
<Avatar.Root>
  <Avatar.Image src="/photo.jpg" alt="Jane Doe" />
  <Avatar.Fallback>JD</Avatar.Fallback>
</Avatar.Root>
```

## Fallback

When the image is unavailable, the fallback renders initials.

<AvatarFallback />

```tsx
<Avatar.Root>
  <Avatar.Fallback>JD</Avatar.Fallback>
</Avatar.Root>
```

## Parts

| Part | Description |
|------|-------------|
| `Root` | Container |
| `Image` | User photo (hides on error) |
| `Fallback` | Shown when image is unavailable |
