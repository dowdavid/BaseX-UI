
## Usage

<CheckboxBasic />

```tsx
<label>
  <Checkbox.Root>
    <Checkbox.Indicator />

</label>
```

<CheckboxStates />

## Parts

| Part | Description |
|------|-------------|
| `Root` | The checkbox control |
| `Indicator` | Check or minus icon |

## Props

### Root

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `checked` | `boolean` | — | Controlled checked state |
| `defaultChecked` | `boolean` | `false` | Initially checked |
| `onCheckedChange` | `(checked) => void` | — | Called on change |
| `indeterminate` | `boolean` | `false` | Indeterminate state |
| `disabled` | `boolean` | `false` | Disabled |
| `required` | `boolean` | `false` | Required for form validation |
