## Import

```tsx
import { Dialog, Button, Field, Input } from '@basex-ui/components';
```

## Usage

<DialogBasic />

```tsx
const [open, setOpen] = useState(false);

<Dialog.Root open={open} onOpenChange={setOpen}>
  <Dialog.Trigger render={<Button variant="outline">Edit profile</Button>} />
  <Dialog.Portal>
    <Dialog.Backdrop />
    <Dialog.Popup>
      <form onSubmit={(e) => { e.preventDefault(); setOpen(false); }}>
        <Dialog.Header>
          <Dialog.Title>Edit profile</Dialog.Title>
          <Dialog.Description>Update your display name and bio.</Dialog.Description>
        </Dialog.Header>
        <Dialog.Panel>
          <Field.Root>
            <Field.Label>Display name</Field.Label>
            <Input type="text" defaultValue="Dave Dow" />
          </Field.Root>
          <Field.Root>
            <Field.Label>Email</Field.Label>
            <Input type="email" defaultValue="dave@ghostdigital.co" />
          </Field.Root>
          {/* Location, Website fields */}
        </Dialog.Panel>
        <Dialog.Footer>
          <Dialog.Close render={<Button variant="ghost">Cancel</Button>} />
          <Button type="submit">Save</Button>
        </Dialog.Footer>
      </form>
    </Dialog.Popup>
  </Dialog.Portal>
</Dialog.Root>
```

## Parts

| Part | Description |
|------|-------------|
| `Root` | Controls open state |
| `Trigger` | Opens the dialog |
| `Portal` | Renders outside the DOM tree |
| `Backdrop` | Overlay |
| `Popup` | Dialog container (includes close button by default) |
| `Header` | Top section |
| `Title` | Heading |
| `Description` | Supporting text |
| `Panel` | Scrollable content area |
| `Footer` | Bottom section for actions |
| `Close` | Closes the dialog |

## Props

### Root

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `open` | `boolean` | — | Controlled open state |
| `defaultOpen` | `boolean` | `false` | Initially open |
| `onOpenChange` | `(open) => void` | — | Called on change |
| `dismissible` | `boolean` | `true` | Close on backdrop click or Escape |

### Popup

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `showCloseButton` | `boolean` | `true` | Show the X close button |

### Footer

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `'default' \| 'bordered'` | `'default'` | Footer style |
