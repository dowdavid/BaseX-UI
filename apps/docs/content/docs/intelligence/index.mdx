Most component libraries give you docs and hope for the best. Base-X UI ships an intelligence layer that sits between the agent and the components. It resolves natural language to component recommendations, catches anti-patterns before code is generated, and provides animation presets so agents never invent their own timing.

The intelligence layer powers the MCP server, but it's also available as a standalone package.

```bash
pnpm add @basex-ui/intelligence
```

```tsx
import { resolveIntent, checkUsage } from '@basex-ui/intelligence';
```

## How it works

An agent describes what it's building. The intelligence layer scores that description against 61 intent mappings, each tied to a specific component with signal keywords, reasoning, and a composition blueprint. The highest-scoring match becomes the recommendation.

Then it runs the description through 70 anti-pattern rules to catch misuse. If the agent is about to use a Button for navigation or an Accordion for a wizard flow, the system warns it before any code is written.

```
Description: "FAQ section with expandable answers"
        │
        ▼
┌─────────────────────┐
│  Intent Resolution   │──▶ Accordion (score: 3)
│  61 intent mappings  │    reasoning: "classic accordion pattern"
└─────────────────────┘    composition: <Accordion.Root>...
        │
        ▼
┌─────────────────────┐
│  Anti-Pattern Check  │──▶ No warnings
│  70 rules            │
└─────────────────────┘
        │
        ▼
  Agent generates code
  with correct component,
  imports, and structure
```

## What this prevents

Agents hallucinate. They pick the wrong component, guess at prop names, invent import paths, and skip required CSS. The intelligence layer eliminates these failure modes:

- **Wrong component** — `resolveIntent` maps intent to component with reasoning. The agent doesn't choose, the system recommends.
- **Wrong props** — `get_component_setup` surfaces required props. The agent never skips `value=` on Accordion.Item.
- **Wrong imports** — Every recommendation includes the exact import statement.
- **Wrong animation** — 5 presets cover every interaction. No magic numbers.
- **Misuse** — 70 anti-patterns catch mistakes like using Dialog for alerts or Checkbox for exclusive selection.
